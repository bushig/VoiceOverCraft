# ❓ Открытые вопросы и заметки

1. **Инициализация перегенерации:** Как фиксировать параметры/версию конфига на момент генерации?
2. **Контроль качества:** Лайки/дизлайки из аддона? Форма обратной связи на фронтенде?
3. **Выбор версии для пака:** Используется `is_active` или сортировка по `updated_at`. Аппрув отдельно для каждого пола.
4. **Принудительная генерация:** Возможность запустить генерацию реплики с конкретным `voice_id`.
5. **Инвалидация озвучек:** Массовый сброс через SQL-запросы по критериям (например, все фразы определенного NPC).
6. **Расхождения текстовок в ядрах:** Критично для `gossip`. План: сравнивать базы версий аддонов, при различиях сохранять отдельные хеши для кросс-совместимости.
