# Backend: AI-озвучивание для World of Warcraft

## 📋 Обзор

Бекенд отвечает за генерацию озвучивания квестов и фраз в WoW через AI.

**Основные задачи:**

- Хранение фраз для озвучки
- Управление голосами и воркерами генерации
- Предоставление API для фронтенда (мониторинг и анализ)

**Наполнение и деплой:**

- `v1`: Выгрузка текстов из `cmangos` (Vanilla)
- `Дальше`: Выгрузка из других ядер приватных WoW
- Первичное наполнение БД → скрипты
- Развертывание → Ansible
