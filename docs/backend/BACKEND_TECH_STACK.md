# 🛠 Технологический стек

## Backend

- **Framework:** FastAPI
- **Database Migrations:** Alembic
- **Database:** PostgreSQL (основное хранилище данных)

## Infrastructure

- **Deployment:** Ansible
- **File Storage:** Хранилище аудио-архивов и промежуточных данных

## Worker

- **Platform:** Windows
- **Workload:** GPU/CPU-intensive задачи генерации

## Frontend

- **Purpose:** Прослушивание результатов, мониторинг статусов генераций

## Addon

- **Platform:** World of Warcraft
- **Purpose:** Работа с архивами озвучки, взаимодействие с API для оценки/голосования
