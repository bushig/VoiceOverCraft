# 🏗 Архитектура

1. **Бекенд:** FastAPI + Alembic + БД (данные, роутинг задач для воркеров/фронтенда)
2. **Хранилище файлов:** Аудио-архивы и промежуточные данные
3. **Фронтенд:** Прослушивание результатов, мониторинг статусов генераций
4. **Воркер (Windows):** GPU/CPU-intensive задачи, поэтапная генерация
5. **Аддон WoW:** Работа с архивами озвучки, взаимодействие с API для оценки/голосования
