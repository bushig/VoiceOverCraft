# 🔄 Рабочий процесс (Flow)

1. В БД загружаются голоса, референсы для AI-клонирования и реплики.
2. Система определяет реплики без генерации и создает задачу.
3. Воркер берет задачу в очередь и выполняет этапы генерации.
4. По завершении формируется архив озвучек. В архив попадают реплики с `is_active=true` или с самой свежей `updated_at`. При необходимости двух гендеров генерируются оба (или `male` как стандарт).
5. Игрок подключает архив как аддон.
6. При низком качестве озвучки игрок ставит дизлайк (аддон/фронтенд) или загружает другой голос для перегенерации.
7. Новая озвучка попадает в следующую выгрузку (если нет явного `is_active`), цикл повторяется.
